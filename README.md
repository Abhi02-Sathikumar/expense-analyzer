A simple, beginner-friendly data analytics project built using Excel and Google Sheets.
It analyzes personal expenses, generates insights, and presents them in a clean dashboard with charts.

Perfect for demonstrating skills in:
✔ Excel formulas
✔ Data analysis
✔ Dashboard creation
✔ Google Sheets basics
✔ Clean data organization

 How to Use

Download expenses_sample.csv

Open Expense_Analyzer.xlsx

Explore:

Summary sheet

Automatic category totals

Dashboard chart

Upload the CSV to Google Sheets to create your online version

Add your Google Sheets link to google-sheets/link.txt
